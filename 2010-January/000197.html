Hello,

Ross and I decided to move the IET subversion repository to
sourceforge.net, where the IET web- and project sites are already hosted
(IET went to berlios.de in the first place when sf.net didn't support
svn).

The subversion repository is now available at:
https://iscsitarget.svn.sourceforge.net/svnroot/iscsitarget

https://sourceforge.net/projects/iscsitarget/develop has a brief
overview on how to use it plus pointers to more documentation.

In addition to that, a new mailing list has been created where commits
to the repository can be monitored:
https://sourceforge.net/mailarchive/forum.php?forum_name=iscsitarget-commits

The old repository + commit list will be kept alive (with new commits
being also sent to the old commit list, but of course no new commits
going to the old repo) for a transition period of 4 weeks to allow users
to update their workspaces and mailing list subscriptions.


The IET development team
