Log Message:
-----------
IET doesn't implement Time2Wait or Time2Retain yet, so negotiating
any other values then zero for these makes no sense and might even
give initiators fits when trying to perform session reinstatement.

This will silently filter any configured value to zero.

Compile tested on CentOS5
